# occasioner-event-service-platform
Full-stack event service booking platform built with PHP, MySQL, and JavaScript.

## Tech Stack

- Frontend: HTML, CSS, JavaScript  
- Backend: PHP  
- Database: MySQL  
- Design: Figma  

---

## Features Overview

- User-friendly interface for browsing event services  
- Service booking system for professionals (chefs, cleaners, photographers, etc.)  
- Structured database for storing services and booking data  
- Responsive design for multiple devices
